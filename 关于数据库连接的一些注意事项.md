# 防止SQL注入攻击的方法：
### 1.将需要引用的变量变为查询参数:
###### 原代码：
`select age from T_student where name ='"+txbname.text+"'"
###### 这行代码在输入的变量的值为 1' or '1= 1时会查询出所有的年龄数据

###### 更改后的代码为:
```
getcmd.commandtext = select age from T_student where name ="@name"
getcmd.Parameters.add(new SqlParameter ("@name",txbname.text))
```
